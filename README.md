Azure Virtual Desktop において ユーザーごとの任意のセッションホストに対する稼働時間かつセッションホストのある時間幅の起動時間を可視化。
またユーザーのあるセッションホスト上での稼働時間を ユーザー稼働時間/セッションホスト稼働時間で導出。
